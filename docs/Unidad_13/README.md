# Unidad 13: Componentes y Modularidad

## Introducción

En el desarrollo profesional, rara vez escribimos todo desde cero. Esta unidad se centra en cómo integrar componentes de terceros (librerías) y cómo estructurar nuestro propio código en módulos reutilizables.
