# Unidad 12: Vistas

## Introducción

La capa de Vista es la cara visible de nuestra aplicación. En esta unidad final del bloque MVC, aprenderemos a gestionar la presentación de datos de manera limpia y separada de la lógica de negocio. Implementaremos un motor de vistas simple pero potente que nos permitirá renderizar HTML dinámico, pasar datos desde el controlador de forma segura y utilizar layouts maestros para mantener la consistencia visual.

## Contenidos

1.  **[Controlador de Vistas](./01_Controlador_Vistas/README.md)**
    *   Definiremos el rol estricto de la vista en POO.
    *   Crearemos la clase `View` para gestionar el paso de variables y el renderizado de archivos.
2.  **[Maquetación](./02_Maquetacion/README.md)**
    *   Aprenderemos a estructurar archivos de vista (HTML/PHP).
    *   Implementaremos layouts principales para reutilizar cabeceras y pies de página.
    *   Veremos el ciclo completo de una petición MVC desde el navegador hasta la respuesta final.
