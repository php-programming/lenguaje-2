# 12.1 Definición de Vistas en POO

Finalmente, llegamos a la capa de presentación: la Vista. Esta es la parte de la aplicación que interactúa directamente con el usuario, mostrando los datos procesados por el Controlador y obtenidos del Modelo.

## Concepto Teórico: El Rol de la Vista

La Vista tiene una sola responsabilidad: presentar información.

- **No contiene lógica de negocio**: La Vista nunca debe consultar la base de datos ni tomar decisiones de lógica compleja.
- **Recibe los datos del Controlador**: Solo imprime las variables y estructuras de datos que le son explícitamente enviadas.
- **Es puramente Presentación**: Está compuesta principalmente por código HTML, y ocasionalmente PHP simple (para bucles `foreach` e impresiones `echo`) y assets (CSS/JS).
