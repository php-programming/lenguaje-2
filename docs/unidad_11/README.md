# Unidad 11: URL y URI

## Introducción

En el desarrollo web, la gestión correcta de las direcciones (URLs) es crucial para la navegabilidad, el SEO y la portabilidad de la aplicación. En esta unidad, formalizaremos los conceptos de URI y URL y construiremos una herramienta (Helper) para gestionar la generación de enlaces de manera centralizada y profesional.

## Contenidos

1.  **[URL y URI](./01_URL_URI/README.md)**
    *   Diferenciaremos técnicamente entre URI (Identificador) y URL (Localizador).
    *   Implementaremos la clase `HelperUrl` para generar enlaces absolutos dinámicamente, facilitando el despliegue en diferentes entornos (local, producción, subcarpetas).
