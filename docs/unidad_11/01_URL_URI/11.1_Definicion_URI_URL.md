# 11.1 Definición de URI y URL

## Concepto Teórico: URI vs. URL

Aunque a menudo se usan indistintamente, URI y URL tienen una distinción técnica importante:

- **URI (Uniform Resource Identifier - Identificador Uniforme de Recurso)**: Es una cadena de caracteres que identifica de manera única cualquier recurso (un documento HTML, una imagen, un servicio, un correo electrónico, etc.). Es la forma más general de identificar algo.
  - Ejemplo: `mailto:contacto@dominio.com` (Identifica una dirección de correo).
  - Ejemplo: `/productos/123` (Identifica el recurso "producto 123" dentro del servidor).
- **URL (Uniform Resource Locator - Localizador Uniforme de Recurso)**: Es un tipo de URI que, además de identificar el recurso, especifica cómo acceder a él y dónde está ubicado. Es la forma más específica.
  - Ejemplo: `https://www.dominio.com/productos/123`

> [!NOTE]
> Una URL siempre es una URI, pero una URI no siempre es una URL (ej. el ejemplo de `mailto:`).

En nuestro contexto web, nos enfocaremos en generar URLs limpias (amigables) a partir de las URIs internas de nuestra aplicación.
