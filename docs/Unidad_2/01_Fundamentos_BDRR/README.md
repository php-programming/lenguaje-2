# Sección 1: Fundamentos de Bases de Datos Relacionales (BDRR)

Antes de poder optimizar un sistema (mediante la normalización), debemos comprender sus componentes fundamentales. Esta sección establece el vocabulario y los principios del modelo relacional, la base teórica sobre la que se construyen sistemas como MySQL, PostgreSQL y SQL Server.

## Contenidos

1.  **[El Modelo Relacional y sus Componentes](./1.1_Modelo_Relacional.md)**: Se definen formalmente los bloques de construcción: Entidades, Atributos, Tuplas y, crucialmente, el concepto de Claves (Primaria, Foránea, Candidata).
2.  **[Reglas de Integridad de Datos](./1.2_Integridad_de_Datos.md)**: Se exploran las garantías que ofrece un SGBDR (Sistema Gestor de Bases de Datos Relacionales): la Integridad de Entidad, la Integridad Referencial y la Integridad de Dominio.
