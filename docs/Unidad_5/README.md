# Unidad 5: Ciberseguridad Aplicada (DevSecOps)

## Introducción

En el entorno actual, la seguridad no es una "funcionalidad extra", es un requisito fundamental. Un ingeniero que no escribe código seguro está creando vulnerabilidades, no soluciones. Esta unidad cubre los principios de defensa en profundidad para aplicaciones web.

2.  **[Prevención de Ataques Comunes](./02_Prevencion_Ataques/5.2_XSS_SQLi.md)**
    *   Defensa contra XSS mediante Output Encoding.
    *   Defensa contra SQL Injection mediante Sentencias Preparadas.
3.  **[Criptografía Aplicada](./03_Criptografia_Aplicada/5.3_Hashing.md)**
    *   Diferencia entre Hashing y Encriptación.
