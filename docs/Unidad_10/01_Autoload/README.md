# Sección 1: Autoload

El Autoloading es una característica esencial de PHP moderno que permite la carga bajo demanda de las definiciones de clases. Esta sección explica cómo configurar un autoloader robusto que siga la estructura de directorios de nuestro proyecto MVC.

## Contenidos

1.  **[Configuración de Autoload](./10.1_Autoload_Config.md)**: Explicación de `spl_autoload_register()`, implementación del script de carga automática y su integración en el punto de entrada `index.php`.
