# Sección 3: Rutas Personalizadas

Para aplicaciones profesionales, el enrutamiento basado solo en convenciones es insuficiente. Esta sección expande nuestro Router para soportar una tabla de rutas personalizada, ofreciendo control total sobre las URLs de la aplicación.

## Contenidos

1.  **[Rutas Personalizadas](./10.3_Rutas_Personalizadas.md)**: Implementación de un sistema de mapeo de rutas explícito, permitiendo URLs semánticas, parámetros dinámicos y desacoplamiento entre la URL pública y el controlador interno.
