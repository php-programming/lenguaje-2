# Unidad 4: Lógica de Programación y Algoritmos

## Introducción

Escribir código es fácil; diseñar lógica eficiente y a prueba de fallos es ingeniería. En esta unidad, nos centraremos en el control de flujo: cómo hacer que nuestro software tome decisiones y automatice tareas repetitivas.

## Contenidos

1.  **[Estructuras de Control Condicionales](./01_Control_de_Flujo/4.1_Condicionales.md)**
    *   Implementaremos lógica de negocio compleja usando `if`, `switch` y operadores ternarios.
2.  **[Algoritmos Iterativos (Ciclos)](./02_Algoritmos_Iterativos/4.2_Ciclos.md)**
    *   Automatizaremos el procesamiento de datos con `for`, `while` y `foreach`.
    *   Analizaremos la complejidad algorítmica básica para evitar problemas de rendimiento.
