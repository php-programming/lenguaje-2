# 7.6 Traits: La Caja de Herramientas

## El "Qué" (Reutilización Horizontal)

Un **Trait** es un mecanismo para reutilizar código horizontalmente. Es, en esencia, "copiar y pegar" un grupo de métodos dentro de una clase en tiempo de compilación.

## El "Por Qué" (Limitaciones de la Herencia)

PHP tiene herencia simple: una clase solo puede tener un padre (`extends`).
¿Qué pasa si `Perro` (hijo de `Mamifero`) y `Robot` (hijo de `Maquina`) necesitan ambos la capacidad de `geolocalizarse`? No pueden heredar de una clase común `GPS`.

**Solución:** Creamos un Trait `Localizable` y ambos lo usan.

## Ejemplo Práctico

```php
<?php
// EL TRAIT
trait Loggable {
    public function log($msg) {
        echo "[LOG]: $msg\n";
    }
}

// CLASE 1
class Usuario {
    use Loggable; // "Copia" el método log() aquí
}

// CLASE 2
class Pedido {
    use Loggable; // También lo usa
}

$u = new Usuario();
$u->log("Usuario creado"); // Funciona gracias al Trait
?>
```
