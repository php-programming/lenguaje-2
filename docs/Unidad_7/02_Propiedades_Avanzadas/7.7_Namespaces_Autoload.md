# 7.7 Namespaces y Autoload: Organización Moderna

## Namespaces (Carpetas Virtuales)

**Problema:** Conflicto de nombres. Tienes tu clase `Database` y descargas una librería que también tiene una clase `Database`. PHP lanzará un error fatal.
**Solución:** Los Namespaces son como apellidos o rutas de carpetas virtuales.
*   Tu clase: `MiApp\Database`
*   Librería: `Vendor\Lib\Database`

```php
namespace App\Models;

class Usuario { ... }
```

## Autoload (Carga Automática)

**Problema:** Tener que escribir 50 `require_once` al inicio de tu script.
**Solución:** Un Autoloader (usualmente gestionado por **Composer**) que intercepta la llamada a una clase y carga el archivo automáticamente basándose en su Namespace (estándar PSR-4).

```php
// index.php
require 'vendor/autoload.php'; // Una sola línea carga todo

use App\Models\Usuario;

$user = new Usuario(); // El autoloader busca src/Models/Usuario.php y lo incluye.
```
