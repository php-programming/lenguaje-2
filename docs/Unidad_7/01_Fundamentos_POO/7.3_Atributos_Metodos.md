# 7.3 Atributos y Métodos: Estado y Comportamiento

Ahora llenemos ese molde vacío. Un objeto se compone de dos cosas: lo que *es* (estado) y lo que *hace* (comportamiento).

## Atributos (Propiedades)

Los atributos son las características o datos de una clase. Son los sustantivos o adjetivos que describen el **estado** de un objeto. Se definen como variables dentro de la clase.

*   **Ejemplos para Coche:** `$marca`, `$color`, `$velocidadActual`.
*   **Ejemplos para Usuario:** `$nombre`, `$email`, `$password`.

## Métodos

Los métodos son las acciones o comportamientos que una clase puede realizar. Son los verbos. Definen la **lógica** del objeto. Se definen como funciones dentro de la clase.

*   **Ejemplos para Coche:** `arrancar()`, `frenar()`, `acelerar()`.
*   **Ejemplos para Usuario:** `login()`, `registrar()`, `cambiarPassword()`.

## Ejemplo Práctico (El Código)

```php
<?php
class Coche {
    // --- ATRIBUTOS ---
    // (public significa que se puede acceder desde fuera)
    public $marca = "Generica"; // Un valor por defecto
    public $color = "Blanco";
    public $velocidadActual = 0;

    // --- MÉTODOS ---
    // Dentro de la clase, usamos '$this' para referirnos
    // a los atributos y métodos de *este* objeto en particular (la instancia actual).

    public function arrancar() {
        echo "El coche $this->marca de color $this->color está arrancando...\n";
    }

    public function acelerar($cuanto) {
        $this->velocidadActual = $this->velocidadActual + $cuanto;
        echo "Acelerando a $this->velocidadActual km/h\n";
    }
}

// --- USANDO LOS OBJETOS ---
$miFerrari = new Coche();
$miFerrari->marca = "Ferrari"; // Modificamos sus atributos
$miFerrari->color = "Rojo";

$tuTwingo = new Coche();
$tuTwingo->marca = "Renault";
$tuTwingo->color = "Azul";

// Llamamos a sus métodos
$miFerrari->arrancar(); // Salida: El coche Ferrari de color Rojo está arrancando...
$tuTwingo->arrancar();  // Salida: El coche Renault de color Azul está arrancando...

$miFerrari->acelerar(100); // Salida: Acelerando a 100 km/h
?>
```
