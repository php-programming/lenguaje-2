# 7.2 Clases y Objetos: El Molde y la Instancia

Esta es la distinción conceptual más importante de toda la unidad. Si entiendes esto, entiendes POO.

## Clase (El Molde)

Una **Clase** es la plantilla, el plano o el molde para crear objetos. No es nada "real" todavía; no ocupa memoria para datos específicos. Es solo la definición abstracta de cómo será algo.

*   Define qué **atributos** (datos) tendrá.
*   Define qué **métodos** (comportamientos) tendrá.

**Analogía:** La clase es el `PlanoDeUnaCasa`. Define que la casa tendrá "3 habitaciones", "2 baños" y "1 cocina". También define que tendrá los comportamientos `abrirPuerta()` y `encenderLuces()`. Pero no puedes vivir en el plano.

## Objeto (La Instancia)

Un **Objeto** es la **instancia** de una clase. Es el producto "real" construido a partir del molde. Puedes crear tantos objetos como quieras a partir de una sola clase, y cada uno tendrá su propia vida.

**Analogía:** Si `PlanoDeUnaCasa` es la clase, los objetos son:
*   `$miCasa` (que es de color verde, en la calle A).
*   `$tuCasa` (que es de color azul, en la calle B).
*   `$laCasaBlanca` (que es blanca, en Washington).

Todos son "Casas" (vienen de la misma clase), pero son instancias únicas e independientes.

## Ejemplo Práctico (El Código)

```php
<?php
// 1. DEFINICIÓN DE LA CLASE (El Molde)
// Convención: Nombre de la clase siempre en 'UpperCamelCase' (PascalCase)
class Coche {
    // (Por ahora, vacía)
}

// 2. CREACIÓN DE OBJETOS (Las Instancias)
// Usamos la palabra clave 'new' para instanciar
$miFerrari = new Coche();
$tuTwingo = new Coche();

// $miFerrari y $tuTwingo son dos objetos distintos en memoria,
// aunque ambos fueron creados con el mismo molde 'Coche'.
var_dump($miFerrari);
var_dump($tuTwingo);
?>
```
