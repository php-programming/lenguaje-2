# Sección 3: JSON (JavaScript Object Notation)

Esta sección aborda la comunicación de datos en aplicaciones modernas. JSON es el formato universal para el intercambio de información entre el servidor y clientes diversos (navegadores, apps móviles, otros servidores).

## Contenidos

1.  **[Concepto de JSON](./9.5_Concepto_JSON.md)**: Definición, estructura y comparación con arrays de PHP.
2.  **[Implementación en PHP](./9.6_Implementacion_PHP.md)**: Guía práctica para usar `json_encode()` y `json_decode()` en controladores para crear APIs RESTful.
