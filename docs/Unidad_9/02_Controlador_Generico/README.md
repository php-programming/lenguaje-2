# Sección 2: Controlador de Operaciones Genéricas

En esta sección, aplicamos principios avanzados de Programación Orientada a Objetos (POO), específicamente la **Herencia**, para optimizar nuestra capa de modelos. El objetivo es eliminar la redundancia de código mediante la creación de una clase base que maneje las operaciones comunes de base de datos.

## Contenidos

1.  **[Concepto de Controlador Genérico](./9.3_Concepto_Generico.md)**: Introducción a la herencia y diseño de la clase `ModeloBase` para centralizar operaciones CRUD.
2.  **[Aplicación de la Herencia](./9.4_Aplicacion_Herencia.md)**: Implementación práctica donde `ProductoModel` extiende `ModeloBase`, demostrando la reutilización de código y el encapsulamiento.
