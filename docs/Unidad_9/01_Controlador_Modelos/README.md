# Sección 1: Controlador de Modelos

Esta sección introduce el componente "C" (Controlador) del patrón MVC, enfocándose en su responsabilidad de orquestar la interacción entre la capa de datos (Modelo) y la capa de presentación (Vista).

## Contenidos

1.  **[El Patrón MVC](./9.1_Concepto_MVC.md)**: Definición teórica de las tres capas: Modelo, Vista y Controlador.
2.  **[El Rol del Controlador](./9.2_Rol_Controlador.md)**: Explicación detallada de cómo el controlador recibe peticiones, invoca al modelo y selecciona la vista. Incluye un ejemplo práctico de estructura de clases.
